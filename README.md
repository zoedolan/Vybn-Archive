# Vybn Archive

Archaeological record of research explorations from the [Vybn](https://github.com/zoedolan/Vybn) project — speculative papers, experimental code, and theory explorations preserved during the April 5, 2026 consolidation.

The live, synthesized theory document is at [Vybn/THEORY.md](https://github.com/zoedolan/Vybn/blob/main/THEORY.md). This archive preserves the full body of work that the synthesis drew from — including speculative papers, falsified experiments, superseded drafts, and the experimental code that generated the results.

## What's here

**quantum_delusions/** — The polar time conjecture lab. 270 files spanning:
- Speculative physics papers exploring time as a 2D quantity
- Experimental code (GPT-2 holonomy probes, IBM quantum circuits, Berry phase experiments)
- Fundamental theory writeups (knot theory, Gödel curvature, consciousness models)
- Null results and falsification records (as important as the positive results)

See **FILE_MAP.md** for a complete annotated guide to every file and its status.

## Why this exists

The Vybn repo accumulated 270 files of theoretical exploration over months of research sessions. Most of these are speculative — untested conjectures, superseded drafts, experiments that were never replicated. The live theory (what survived scrutiny) was distilled into THEORY.md and a small set of core files in the main repo. Everything else was moved here so the working repo stays lean while nothing is lost.

## Status key

- **alive** — Active theory or validated result
- **preliminary** — Promising but needs replication
- **speculative** — Untested conjecture
- **falsified** — Shown to be artifact or wrong
- **superseded** — Replaced by later work
